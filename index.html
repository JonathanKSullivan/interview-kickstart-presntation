Functional Programming in Python

Diving into map, filter, and reduce for efficient data manipulation.

map()

Applies a function to all items in an input list.


				# Example: Square numbers in a list
				numbers = [1, 2, 3, 4]
				squared = list(map(lambda x: x**2, numbers))
				print(squared)

Expected Output: [1, 4, 9, 16]

Functional programming enhances our data handling through pure functions and immutable data. The map function, for instance, applies a transformation across all items in a collection, ideal for calculating delivery times. Let’s see this in practice in our next live coding session. Functional Programming in Python

Diving into map, filter, and reduce for efficient data manipulation.

filter()

Creates a list of elements for which a function returns true.


				# Example: Filter even numbers
				numbers = [1, 2, 3, 4, 5, 6]
				even = list(filter(lambda x: x % 2 == 0, numbers))
				print(even)

Expected Output: [2, 4, 6]

The filter function allows us to sift through our data, selecting items that meet specific criteria, such as battery level or availability. This is particularly useful in preparing data for machine learning models. Let’s move to a live demo to see filter in action. reduce()

Performs a computation on a sequence of values to reduce it to a single value.


				# Example: Sum numbers in a list
				from functools import reduce
				numbers = [1, 2, 3, 4]
				result = reduce(lambda x, y: x + y, numbers)
				print(result)

Expected Output: 10
